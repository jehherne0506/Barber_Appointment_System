<h2>🧐 Features</h2>

Here're some of the project's best features:

*   User Authentication
*   Service & Barber Management
*   Appointment Booking System
*   Appointment History
*   Email Notifications
*   Dashboard & Statistics
*   Admin / Staff Controls
*   Responsive UI
